## 1) 플롯 옵션 A: (강한 공감/문제 폭로형) **“아침 루틴, 왜 매번 실패해?” 글라스 스킨 듀오**
### A-1. 목표/타깃/핵심 메시지
- 타깃: 글라스 스킨/모닝 루틴 관심층, 루틴 따라 구매하는 초보 스킨케어 유저
- 한 문장 메시지: “글라스 스킨, 루틴이 복잡해서 못 하겠다면 ‘2종으로 끝’.”
- 기대 반응(시청자 감정/행동): “나도 저거다(공감) → 간단해 보인다(안심) → 링크 클릭/장바구니”

### A-2. 15초 버전 타임라인(초 단위)
- 0~3s (HOOK): **[온스크린 텍스트]** “글라스 스킨? 3분만에 ‘딱 2개’로 끝” / **[화면]** 윤광 클로즈업(전면 카메라) + 손에 2개 제품만 들기 / **[대사(VO)]** “복잡한 루틴? 오늘은 2개만.”
- 3~6s (PROBLEM): **[텍스트]** “단계 많으면… 결국 안 함😵” / **[화면]** 욕실 선반에 제품 잔뜩 → 화면 스와이프 컷 / **[VO]** “아침엔 시간 없잖아.”
- 6~10s (SOLUTION/DEMO): **[텍스트]** “1) 모이스처라이저 2) 미스트” / **[화면]** 바르는 손동작 + 미스트 분사(입자 반사) / **[VO]** “바르고, 뿌리면 끝.”
- 10~13s (PROOF/REASON): **(보고서 근거 기반)** **[텍스트]** “이 프레임이 요즘 ‘glass skin / routine’로 뜸” / **[화면]** #morningroutine #glassskin 태그를 ‘자막’으로만 노출(수치 추가 금지) / **[VO]** “요즘 다 ‘루틴’으로 보여주더라.”
- 13~15s (CTA): **[텍스트 크게]** “프로필 링크 눌러서 ‘루틴 듀오’ 확인” / **[화면]** 손가락으로 프로필 링크 방향 가리키기 / **[VO]** “프로필 링크, 지금 확인!”

### A-3. 25~35초 버전 타임라인(초 단위)
- 0~3s (HOOK): **[텍스트]** “글라스 스킨… 근데 루틴은 못 지키겠어?” / **[화면]** 반짝 윤광 클로즈업 → 바로 컷 전환
- 3~8s (PROBLEM 확장): **[텍스트]** “단계 많으면 ‘선택 피로’로 포기” / **[화면]** ‘제품 7개’ 나열 → “오늘도 실패” 표정 연기 / **[VO]** “나도 루틴만 하다 지쳤어.”
- 8~18s (DEMO/사용장면):  
  - **[텍스트]** “STEP 1 보습(모이스처라이저)” / **[화면]** 텍스처(크림/젤 등은 단정 금지 → ‘보습’만)  
  - **[텍스트]** “STEP 2 미스트(분사)” / **[화면]** 분사 순간 슬로우(Stop Point)  
  - **[VO]** “딱 두 단계로 ‘루틴 느낌’만 살려.”
- 18~28s (근거/차별점 2~3개): **(보고서 근거 기반)**  
  - **[텍스트]** “#glassskin, #skincare routine 상승 프레임”  
  - **[텍스트]** “Moisturisers & Mists는 전환 강신호 카테고리(보고서)”  
  - **[VO]** “보고서 기준으로 ‘모이스처/미스트’ 쪽이 전환이 좋게 관측됐대.”
- 28~35s (CTA + 반박처리/FAQ 1개):

  - **[텍스트 크게]** “프로필 링크에서 세트 확인 → 장바구니”  
  - **[VO]** “스펙은 확인 필요! 일단 링크에서 구성부터 봐.”

- 컷8: 프로필 링크 가리키는 제스처 + CTA 자막
